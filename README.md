# WorkshopExample

Projeto institucional B2B em **ASP.NET Core Razor Pages (.NET 8)** com 3 páginas públicas:

- Home
- Projetos
- Contato

## Escopo

- Sem banco de dados
- Sem autenticação/login
- Formulário de contato com validação e envio simulado
- Layout responsivo com Bootstrap

## Como executar

1. Abra um terminal na pasta `/home/runner/work/WorkshopExample/WorkshopExample/GlobalWeb`
2. Rode:

```bash
dotnet restore
dotnet run
```

3. Acesse a URL exibida no terminal (ex.: `http://localhost:5000`)

## Build

Para validar compilação:

```bash
dotnet build /home/runner/work/WorkshopExample/WorkshopExample/GlobalWeb.sln
```
